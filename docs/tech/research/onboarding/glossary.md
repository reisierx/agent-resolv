# Glossary -- Portuguese Insurance Terms

> **Version:** 1.0
> **Last updated:** March 2, 2026
> **Status:** Draft -- internal review

---

| Term                        | Translation / Meaning                                                                           |
| --------------------------- | ----------------------------------------------------------------------------------------------- |
| ASF                         | Autoridade de Supervisao de Seguros e Fundos de Pensoes -- Portugal's insurance regulator       |
| BdP                         | Banco de Portugal -- central bank, regulates credit intermediaries                              |
| Mediador de seguros         | Insurance mediator (agent) -- tied to one or few insurers                                       |
| Corretor de seguros         | Insurance broker -- independent, represents the customer, not the insurer                       |
| NIF                         | Numero de Identificacao Fiscal -- Portuguese tax identification number (9 digits)               |
| Apolice                     | Insurance policy document                                                                       |
| Premio                      | Insurance premium (the price paid)                                                              |
| Franquia                    | Deductible / excess -- amount the customer pays before insurance kicks in                       |
| Sinistro                    | Claim / incident triggering insurance coverage                                                  |
| Responsabilidade civil (RC) | Third-party liability coverage                                                                  |
| Contra todos os riscos      | Comprehensive / all-risk coverage                                                               |
| Danos proprios              | Own-damage coverage (distinct from third-party liability)                                       |
| Bonus-malus                 | No-claims discount/penalty system for auto insurance                                            |
| Capital do edificio         | Building rebuild cost (for home insurance -- differs from market value)                         |
| Capital de recheio          | Contents value (for home insurance)                                                             |
| Caderneta predial           | Property registry document                                                                      |
| Matricula                   | Vehicle license plate number                                                                    |
| Carta de conducao           | Driving license                                                                                 |
| QIS                         | Questionario Individual de Saude -- individual health questionnaire (for health/life insurance) |
| Vida credito                | Credit-linked life insurance (required for mortgages)                                           |
| Multirriscos                | Multi-risk home insurance (combined building + contents)                                        |
| Escritura                   | Notarial deed (used in property/mortgage transactions)                                          |
| DUA                         | Documento Unico Automovel -- vehicle registration document                                      |
| Cilindrada                  | Engine displacement (cc)                                                                        |
| Municipio de circulacao     | Municipality where the vehicle is primarily used                                                |
| DORA                        | Digital Operational Resilience Act (EU regulation for financial sector ICT)                     |
| FiDA                        | Financial Data Access regulation (upcoming EU regulation for open financial data)               |
